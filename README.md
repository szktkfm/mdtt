
## Usage

| Key binding      | Description      |
| ---------------- | ---------------- |
| `Arrows`, `hjkl` | Move cursor      |
| `Enter`   :100:  | Enter directory  |
| `Backspace`      | Exit directory   |
| `Space`          | Toggle preview   |
| **Esc**, _q_     | Exit with cd     |
| `y`              | yank current dir |
